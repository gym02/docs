## How to Use

In your WebSocket connection, specify the voice style in the `session.update` message:

```javascript
socket.send(JSON.stringify({
  type: 'session.update',
  session: {
    instructions: 'You are a helpful assistant.',
    voice: 'echo', // Choose from: alloy, ash, ballad, cedar, coral, echo, marin, sage, shimmer, verse
    // ... other configuration
  }
}));
```
