## Try the Avatars

## How to Use

To use any of these avatars in your WebSocket connection, specify the character name in the connection URL:

```javascript
const CHARACTER_NAME = 'navtalk.Astrid'; // Replace with your chosen avatar
const LICENSE = 'your-license-key';
const VOICE = 'coral'; // Optional: specify voice
const MODEL = 'gpt-realtime'; // Optional: specify model
const websocketUrl = `wss://transfer.navtalk.ai/wss/v2/realtime-chat?license=${encodeURIComponent(LICENSE)}&name=${encodeURIComponent(CHARACTER_NAME)}&voice=${encodeURIComponent(VOICE)}&model=${encodeURIComponent(MODEL)}`;
```

<Note>
All avatars are available immediately after your account is created. No additional setup is required.
</Note>
