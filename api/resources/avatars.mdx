## Try the Avatars

## How to Use

To use any of these avatars in your WebSocket connection, specify the character name in the connection URL:

```javascript
const CHARACTER_NAME = 'navtalk.Astrid'; // Replace with your chosen avatar
const websocketUrl = `wss://transfer.navtalk.ai/api/realtime-api?license=${LICENSE}&characterName=${CHARACTER_NAME}`;
```

<Tip>
All avatars are available immediately after your account is created. No additional setup is required.
</Tip>
