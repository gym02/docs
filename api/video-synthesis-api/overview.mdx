<Hero
  title="Video Synthesis API"
  subtitle="Generate professional digital human videos on demand using images, videos, or preset characters with text-to-speech or audio synchronization"
/>

**NavTalk Video Synthesis API** enables you to generate high-quality digital human videos programmatically. Perfect for creating marketing videos, educational content, personalized messages, and automated video production workflows.

## Core Features

<CardGroup cols={3}>
  <Card
    title="Multiple Input Methods"
    icon="image"
  >
    Support for images, videos, or system preset characters as the visual source with 9 different calling combinations
  </Card>
  <Card
    title="Flexible Audio Options"
    icon="volume-high"
  >
    Use text-to-speech (TTS), audio URLs, or Base64-encoded audio with multiple voice styles for synchronization
  </Card>
  <Card
    title="Asynchronous Processing"
    icon="clock"
  >
    Efficient task-based processing with status polling for scalable video generation
  </Card>
  <Card
    title="High-Quality Output"
    icon="video"
  >
    Professional-grade video synthesis with accurate lip sync and natural expressions
  </Card>
  <Card
    title="System Characters"
    icon="user"
  >
    Use built-in digital human characters or create custom avatars for your brand
  </Card>
  <Card
    title="Easy Integration"
    icon="plug"
  >
    Simple RESTful API with standard JSON requests and responses
  </Card>
</CardGroup>
