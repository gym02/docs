---
title: "session.insufficient_balance"
description: "Account balance insufficient error"
---

Triggered when account balance is insufficient to continue the service. Handle this event to prompt users to recharge their account and provide a clear path to payment.

## Event Properties

<ResponseField name="type" type="string">
Event type. Always `"session.insufficient_balance"` for this event.
</ResponseField>

```json Example
{
  "type": "session.insufficient_balance"
}
```
</ResponseExample>
